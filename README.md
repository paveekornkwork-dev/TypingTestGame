## ‚öôÔ∏è How to Run (Local Development)

### 1. Prerequisites
You only need a modern web browser (Chrome, Firefox, or Edge).

### 2. Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/typing-test-game.git](https://github.com/your-username/typing-test-game.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd typing-test-game
    ```

### 3. Launching the Game
There are two ways to view the project locally:

**Option A: Open directly (Simple)**
* Locate the `index.html` file in your file explorer.
* Double-click it to open it in your default browser.

**Option B: Using Live Server (Recommended for developers)**
* If you use **VS Code**, install the "Live Server" extension.
* Right-click on `index.html` and select **"Open with Live Server"**.

## üéÆ How to Play
1. The game will display a Computer Science term on the screen.
2. Type the word exactly as shown.
3. You have **5 lives**. Each time you fail to complete a word in time or misspell it, you lose a life.
4. Try to get the highest score possible!
