# 🚀 Typing Test Game

A web-based typing game designed to help users practice typing while learning **Computer Science vocabulary**.

The game challenges players to type words accurately before time runs out. Each mistake reduces the player's remaining lives, making the game both educational and engaging.

## ✨ Key Features

- 📚 **Computer Science Vocabulary**  
  Practice typing with curated terms related to Computer Science.

- ❤️ **Life System**  
  Players start with **5 lives**. Each incorrect or missed word costs one life.

- ⌨️ **Typing Practice**  
  Improve typing speed and accuracy through interactive gameplay.

- 🎮 **Simple and Engaging Gameplay**  
  Easy to start and challenging to master.

## ⚙️ How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/typing-test-game.git
```

### 2. Navigate to the Project Folder

```bash
cd typing-test-game
```

### 3. Run the Project

You can run the project in two ways:

#### Option A: Open Directly

Simply open the `index.html` file in your browser.

#### Option B: Use Live Server (Recommended)

If you use **VS Code**:

1. Install the **Live Server** extension
2. Right-click `index.html`
3. Select **Open with Live Server**

---

## 🎮 How to Play

1. A **Computer Science word** will appear on the screen.
2. Type the word **exactly as shown**.
3. You start with **5 lives**.
4. If you misspell a word or run out of time, you lose a life.
5. Try to **achieve the highest score possible**.
